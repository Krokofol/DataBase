Основные сущности :
	- Ареал обитания :
		- ID ареала 	(ключевое поле, целое число, счётчик, обязательное поле)
		- Название 	(короткая строка, обязательное поле)
		- Тип климота	(короткая строка, обязательное поле)
		- Континент	(короткая строка, обязательное поле, список значений : "Евразия", "Африка", "Северная Америка", "Южная Америка", "Австралия", "Антарктида")
	- Вид животного :
		- ID вида	(ключевое поле, целое число, счётчик, обязательное поле)
		- Название вида	(короткая строка, обязательное поле)
		- Тип питания 	(короткая строка, обязательное поле, список значение : "хищник", "травоядное", "всеядное")
		- ID ареала	(целое число, проверка целостности, обязательное поле)
	- Клетка зоопарка :
		- ID клетки 	(ключевое поле, целое число, счётчик, обязательное поле)
		- Длина		(целое число, обязательное поле)
		- Ширина	(целое число, обязательное поле)
		- Высота	(целое число)
		- Расположение	(короткий текст, обязательное поле)
	- Обитатель зоопарка : 
		- ID обитателя  (ключевое поле, целое число, счётчик, обязательное поле)
		- ID вида	(целое число, обязательное поле, проверка целостности)
		- ID клетки	(целое число, обязательное поле, проверка целостности)
		- Кличка	(короткая строка)
		- Вес		(целое число)
		- Длина		(целое число)
		- Дата поступл. (дата)
Основные отношения :
	1. Вид животного 	<->> 	Ареал обитания 	(один ко многим)
	2. Обитатель зоопарка 	<->>	Вид животного	(один ко многим)
	3. Обитатель зоопарка 	<->>	Клетка зоопарка	(один ко многим)
